Git — розподілена система керування версіями файлів.
Git має три основних стани, в яких можуть перебувати файли: збережений у коміті (commited), змінений (modified) та індексований (staged).
Для додавання файла в репозиторій.
Для перевірки чи додані файли (виділені зеленим кольором – додані, червоним – не додані).
Для фіксації стану сховища в певний момент часу.
Для ініціалізації Git у папці проекту.
Описують, що було змінено, де змінено, чому змінено.
git log
git checkout дозволяє переміщатися між гілками, зробленими командой git branch.
